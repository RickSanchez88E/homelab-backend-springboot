## 你的项目地图（先看懂这个！）

你现在运行的这个项目是一个**电商系统**，拆成了 7 个独立小程序：

```
你（浏览器/App）
    ↓ 发送请求
[API Gateway - 9191 端口]  ← 所有请求的入口，像酒店前台
    ↓ 转发到对应服务
┌─────────────────────────────────────┐
│ identity-service (8085) 管用户登录  │
│ product-service  (8081) 管商品      │
│ order-service    (8082) 管订单      │
│ payment-service  (8083) 管支付      │
│ email-service    (8084) 管发邮件    │
└─────────────────────────────────────┘
    ↓ 这些服务都在这里登记注册
[service-registry - 8761 端口]  ← 像通讯录，记录谁在哪
```

每个服务都是一个独立的 Spring Boot 应用。
每个服务都有自己的 MySQL 数据库。
服务之间通过 Kafka 传递消息（比如下单了通知支付服务）。
